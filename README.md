## 环境

```
node @7.9.0
db @3.4.0
...
 the other modules , youcan use "npm install"  to install
 
 attention: after the initiation of MongoDB, you need  initialize a admin count, so that you can manage the backend with it.

```

## run

    git clone git@github.com:Nealyang/React-Express-Blog-Demo.git
    
    npm i
    
    npm start

## build

    npm run build

## run in production environment
    
    npm run start-prod
